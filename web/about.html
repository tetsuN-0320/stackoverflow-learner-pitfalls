About このプロジェクト

データソース

  • Stack Exchange API v2.3
    Stack Overflow の質問・タグデータを取得。 Stack Exchange Network 利用規約 に基づき、 CC BY-SA 4.0 で利用。
  • Stack Overflow Developer Survey
    年次開発者調査データ。 insights.stackoverflow.com/survey より取得。

分析手法

  • マクロ分析: 言語別タグの質問数推移（2015〜2024年）
  • メゾ分析: タグ共起ネットワーク（networkx + Louvain法によるコミュニティ検出）
  • ミクロ分析: 質問本文のトピッククラスタリング（TF-IDF + HDBSCAN）

利用技術

Python / pandas / scikit-learn / networkx / NLTK / Plotly / D3.js

免責事項
